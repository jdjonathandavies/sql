## RDBMS ##

**R**elational **D**atabase **M**anagement **S**ystem. Data is stored in tables. Tables consist of rows and columns.

Constraints are rules enforced on data columns in a table, they can be things like:
* NOT NULL
* DEFAULT
* UNIQUE
* ....
